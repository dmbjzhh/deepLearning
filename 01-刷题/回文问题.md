### 回文问题

* 回文字符串
* 回文整数

我的原先做法

```python
class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        if x < 0:
            return False
        if x % 10 == 0 and x != 0:
            return False
        l = 1
        y = x
        while y // 10 >= 1:
            l += 1
            y = y//10
        f = True
        for i in range(l/2):
            if x // pow(10, i) %10 != x // pow(10, l-i-1) % 10:
                f = False
                break

        return f
```
